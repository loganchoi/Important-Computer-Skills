Post-lab assignment
==============================


Follow "IDE-assignment.pdf" to complete your post-lab

Complete the post-lab by creating/editing the following files in your repository folder:
* Source files like .cpp, .h

Add:
* Create two separate folders for CodeBlocks and Qt-Creator
* Add all the respective files generated from the above two IDEs into their individual folder

Once you are done, upload the files to your git repo.

If you see your changes reflected on the git-classes site, you have submitted successfully.

Git cheat sheet: https://gitlab.com/gitlab-com/marketing/raw/master/design/print/git-cheatsheet/print-pdf/git-cheatsheet.pdf
